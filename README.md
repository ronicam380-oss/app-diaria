# app-diaria

Este é um app Android baseado em Kivy, escrito em Python.

## Configuração

1. Instale as dependências: `pip install -r requirements.txt`

2. Instale o buildozer: `pip install buildozer`

3. Execute no PC: `python main.py`

4. Para construir para Android: `buildozer android debug`

## Substituindo o código

Substitua o conteúdo de `main.py` pelo seu código Python existente. Certifique-se de que seja compatível com Kivy para Android.
